Personal Portfolio Project

Investment Analysis System

個人開発の投資分析システム群を一つに統合したWebポータル。米国市場・日本株式の分析から、UiPathによるデータ自動収集、AIエージェントによるデータメンテナンスまでをワンストップで提供します。

14 Tools / 4 Sections FastAPI + JavaScript AI Chat Assistant UiPath RPA連携
プロジェクト概要

投資分析ポータル — Investment Analysis Portal

統合管理ポータル

米国市場分析・日本株分析・データ収集・データメンテナンスの4セクション、計14個のツールを一つのWebアプリ上で管理。FastAPIをバックエンドに、HTML/CSS/JavaScriptでフロントエンドを構築しています。

AIチャット内蔵

データメンテナンスQ&Aアシスタントを内蔵し、Claude APIを活用したAIエージェントがデータ取込や運用に関する質問に対応します。

RPAによる自動収集

UiPathと連携し、米国・日本の経済指標や日本株価データを自動収集。手作業を排した安定的なデータパイプラインを実現しています。

外部公開ダッシュボード

主要な経済指標分析機能はStreamlitダッシュボードとして切り出し、誰でもアクセスできる形で公開しています。

搭載ツール一覧

4セクション / 14ツール

米国市場分析 3 tools

経済指標分析（暴落監視）
米国の主要経済指標をモニタリングし、市場の暴落リスクを早期に察知する分析ツール。
NYダウ分析
NYダウの価格動向・トレンドを可視化し、相場の転換点を分析。
ドル円分析
USD/JPYの為替動向を分析し、株式市場との連動性を確認。

日本株分析 4 tools

日本版経済指標分析
日本国内の経済指標を集約し、米国版同様の暴落監視・トレンド分析を実施。
株価分析（バフェット式）
バフェット指数等のバリュエーション指標に基づき、個別株の割安・割高を判定。
サヤ取り分析
関連銘柄間の価格差（サヤ）を分析し、ペアトレード機会を検出。
Stock Agent
AIエージェントが銘柄データを解析し、投資判断のヒントを自動生成。
AI AGENT

UiPathデータ収集 3 tools

米国経済指標収集
UiPathロボットが米国経済指標サイトを巡回し、最新データを自動取得。
日本経済指標収集
日本国内の経済指標発表データをRPAで定期収集。
日本株価収集
日本株の価格データをRPAで自動取得し、分析用DBへ蓄積。

データメンテナンス 4 tools

決算プロデータDL
決算プロからの財務データダウンロードと取込を自動化。
前年株価データDL
前年同期の株価データを取得し、比較分析の基礎データを整備。
IRBankデータ取込
IRBankの財務・業績データを取込み、分析用データセットに統合。
EDINET従業員データ取込
EDINET APIを通じて有価証券報告書の従業員データを取込み・蓄積。
